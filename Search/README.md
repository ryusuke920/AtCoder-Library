## Search
|File Name|Algorithm|Explanation|
|:--:|:--:|:--:|
|BinarySearch.py|二分探索|O(logN)で行えるのでとても高速|
|BitSearch.py|Bit全探索|スイッチの ON/OFF など２値価問題に対して使える  計算量は O(2^N)|
|BreadthFirstSearch.py|幅優先探索|迷路の最短経路を考える問題の時などに役に立つ|
